# srs-expense-tracker-v6

Система для обліку особистих витрат та автоматичної генерації звітів.

## Функціонал:
- Облік витрат (модель `Expense`)
- Генерація звітів (наприклад, для ПФУ)
- Аналітика та статистика витрат

## Як використовувати:
1. Запустив `app.py` для старту системи.
2. Програма дозволяє створювати записи про витрати та формувати звіти.
